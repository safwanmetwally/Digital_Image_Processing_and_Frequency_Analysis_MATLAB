# 📊 Fast Fourier Transform (FFT) Analysis using MATLAB

This repository contains a professional MATLAB implementation for analyzing signals in the frequency domain using the *Fast Fourier Transform (FFT)* algorithm. It is designed to transform time-domain signals into their constituent frequencies.

## 🚀 Overview
The project focuses on processing discrete signals to identify dominant frequency components. By converting a signal from the time domain $x(t)$ to the frequency domain $X(f)$, we can filter noise and analyze harmonics.

## 🛠️ Features
* *Signal Generation:* Scripts to generate multi-tone signals.
* *Spectrum Analysis:* Precise calculation of magnitude and phase spectra.
* *Visualization:* High-quality plots including Time-Domain and Amplitude Spectrums.

## 📐 Mathematical Foundation
The Discrete Fourier Transform (DFT) is defined as:
$$X(k) = \sum_{n=0}^{N-1} x(n) \cdot e^{-j \frac{2\pi}{N} nk}$$
